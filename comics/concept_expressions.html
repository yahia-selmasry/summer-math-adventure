I have some cookies.
I'm not saying how many. I'll call it… n.
🍪
n cookies
So n is a mystery number?! Like a variable?
Exactly! Let's learn the key words.
3x + 5
VARIABLE → x
a letter = unknown number
COEFFICIENT → 3
number in front of variable
CONSTANT → 5
plain number, no variable
TERMS → 3x and 5
parts separated by + or −
How to WRITE expressions
"Let n = the unknown number"
"a number plus 8"
n + 8
"5 times a number"
5n
"a number divided by 4"
n ÷ 4
"twice a number minus 3"
2n − 3
To evaluate an expression, just swap the variable for its value and calculate!
Evaluate 3n + 4 when n = 5:
3n + 4
→
3 × 5 + 4
→
15 + 4 = 19
👉 Plug in n = 5, then simplify!
What about 3x + 5x? Can we simplify that?
🔍 LIKE TERMS = same variable → you CAN combine them!
3x + 5x
=
8x
(3 + 5 = 8 x's)
🚫 UNLIKE TERMS = different variables → CANNOT combine!
3x + 5y
≠
8xy ← WRONG!
x and y are different — they stay separate!
Simplify these:
7x + 2x
=
9x
5a − 2a
=
3a
3x + 4 + 2x + 1
=
5x + 5
Group the x's together, then the numbers! 🧩
