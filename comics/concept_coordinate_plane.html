📍 ORDERED PAIRS (x, y)
Every point has an address called an ordered pair!
(x, y) = (right/left, up/down)
• x comes FIRST — go left or right
• y comes SECOND — go up or down
• Always start at the origin (0, 0)
🎯 How to plot (3, 5):
Step 1 Start at the origin (0,0)
Step 2 Move 3 right (x = 3)
Step 3 Move 5 up (y = 5)
Step 4 Put a dot and label it!
Memory trick: "x before y"
Go across the hallway (x), then up the stairs (y)!
🧭 The Four Quadrants
Quadrant x sign y sign Example
I + (right) + (up) (3, 4)
II − (left) + (up) (−2, 5)
III − (left) − (down) (−4, −3)
IV + (right) − (down) (5, −1)
🏛️ Memory trick:
Quadrants go counterclockwise starting top-right.
"All Students Take Calculus" → tells you which trig functions are positive, but for us: I = all positive!
📖 Reading Points Off a Graph
Always read the x first, then the y!
1. Find the point on the graph
2. Draw a line straight down to the x-axis → that's your x
3. Draw a line straight left to the y-axis → that's your y
4. Write as (x, y)
⚠️ Don't confuse (3, 5) with (5, 3) — they are different points!
📏 Distance on the Grid
Points on the same horizontal line (same y):
→ Distance = |x₁ − x₂|

Points on the same vertical line (same x):
→ Distance = |y₁ − y₂|

Example: A(1, 3) and B(5, 3)
Distance = |1 − 5| = |−4| = 4 units

⚠️ WATCH OUT!
❌ (x, y) not (y, x) — x always first!
❌ Up = positive y, Down = negative y
❌ Right = positive x, Left = negative x
❌ Points on an axis are not in any quadrant
❌ The origin is (0, 0) — not (1, 1)!
