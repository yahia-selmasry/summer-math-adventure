Adding integers has
two rules — one for
same signs, one for different!
RULE 1: Same signs → ADD and KEEP the sign
+5 + +3 = +8
both positive → add → positive
−4 + (−2) = −6
both negative → add → negative
RULE 2: Different signs → SUBTRACT and KEEP sign of bigger absolute value
+7 + (−3) = +4
7−3=4, 7 is bigger, so positive
+2 + (−9) = −7
9−2=7, 9 is bigger, so negative
