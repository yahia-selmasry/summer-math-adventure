EXPRESSIONS & EQUATIONS
Key vocabulary: A variable is a letter standing for an unknown number. An expression (like 3x + 5) has no equals sign. An equation (like 3x + 5 = 20) has an equals sign — solve it!
One-step: x + 7 = 15 → x = 8  |  Two-step: 2x + 3 = 11 → 2x = 8 → x = 4
Inequality words: "at least" = ≥, "at most" / "no more than" = ≤, "more than" = >, "fewer than" = <

ANSWER KEY

Page 1 Answers

1. b + 6 = 22; b = 16 apples in bag
2. c − 15 = 48; c = 63 cards to start
3. 5n = 120; n = 24 books per shelf
4. x ÷ 9 = 7; x = 63
5. 8b + 5 = 53; 8b = 48; b = 6 books
6. 12h + 10 = 58; 12h = 48; h = 4 hours
7. 6a + 24 = 114; 6a = 90; a = 15 apples per tree
8. 3n + 2 = 29; 3n = 27; n = 9
Challenge: (4n − 6) ÷ 2 = 11 → 4n − 6 = 22 → 4n = 28 → n = 7

Page 2 Answers

9. h ≥ 48 inches; e.g., 48, 52, 60 inches all qualify
10. 18 + w ≤ 25; w ≤ 7 pounds
11. 147 + b > 200; b > 53; minimum 54 more books
12. 7t ≤ 50 − 8 = 42; t ≤ 6; maximum 6 tickets
13. n=5: 4(5)+3 = 23; n=12: 4(12)+3 = 51
14. h=4,t=3: 8(4)+5(3) = 32+15 = $47; h=6,t=2: 8(6)+5(2) = 48+10 = $58
15. P = 2(14+9) = 46; w = 48/2 − 15 = 24 − 15 = 9
16. s=8,f=3: 5(8)−2(3) = 40−6 = 34; s=12,f=5: 5(12)−2(5) = 60−10 = 50
