ANSWER KEY

Page 1 Answers

1. 47,000 (47,382 → thousands digit 7, next digit 3 < 5, round down)
2. 1,100 miles (1,073 → hundreds digit 0, next digit 7 ≥ 5, round up)
3. 8,600 tickets (8,647 → hundreds digit 6, next digit 4 < 5, round down)
4. 30,000 fans (34,519 → ten-thousands digit 3, next digit 4 < 5, round down)
5. Lakeside (326,140); difference = 280 people
6. Alex (204,750); ahead by 180 points
7. $12,450 + $9,875 = $22,325; $25,000 − $22,325 = $2,675 still needed
8. 38,200 + 4,350 − 2,175 = 40,375 books
Challenge: Legs: 1,200 + 1,000 + 1,500 = 3,700 (est); 1,248 + 976 + 1,534 = 3,758 (exact); difference = 58 miles

Page 2 Answers

9. Estimate: 1,200 + 500 + 700 = 2,400; Exact: 2,377 animals
10. Estimate: 4,000 + 4,000 = 8,000; Exact: 8,011 items

12. Rule: +70 bags/week; Week 6 = 460 + 70 + 70 + 70 = 670 bags (Week 4=530, 5=600, 6=670)

15. Standard: 243,512; Expanded: 200,000 + 40,000 + 3,000 + 500 + 10 + 2
